# Elm Interiors Experience Brief

## Thesis

Make Elm Interiors feel like a calm architectural walkthrough by using warm source interiors, oversized editorial type, and slow room-to-room reveals.

## First Viewport

- Brand signal: an Elm monogram rail, Pretoria studio label, and a full-width kitchen hero.
- Hero scene: source kitchen interior with a material palette overlay and a single project CTA.
- Next-section hint: a horizontal “refined living” process line enters the lower viewport.

## Sequence

1. Full-bleed room hero and project CTA.
2. Material/process rail for nature, planning, and render clarity.
3. Services sequence that moves from architecture to decoration to staging.
4. Portfolio mosaic and owner-led studio story.
5. Contact section with phone, email, location, and project brief prompts.

## Signature Interaction

- Motion map: hero image drifts on scroll; project tiles reveal like staggered room cuts; service copy fades between material bands.
- Reduced motion: keep the hero and tiles static with opacity-only reveals.

## Assets

- Keep: source kitchen, living-space, décor, and portfolio images plus the small source mark.
- Replace: repetitive CMS layout and form chrome with clear static conversion content.
