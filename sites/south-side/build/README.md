South Side Hard Soda rebuild (local dev).

## Getting Started

Install deps and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Routes

- `/` home page for South Side Hard Soda

## Notes

- Rebuild uses a new South Side home experience with hero, flavour cards, product benefits, and store locator CTA.
- Content is based on the South Side site design and brand messaging from the scraped Wix source.
- This scaffold was created by copying the existing Port443 build and updating it for South Side.
