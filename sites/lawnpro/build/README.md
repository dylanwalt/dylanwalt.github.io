# LawnPro local rebuild

Next.js rebuild of the scraped LawnPro source site.

## Run locally

```bash
npm install
npm run dev -- --port 3001
```

Open `http://localhost:3001`.

## Coverage

- Marketing home page
- Services and treatment paths
- Seasonal maintenance and winter program
- Product guide, shop, source product categories, and product-detail routes
- Franchise and contact paths
- Captured utility routes for cart, checkout, privacy, terms, and returns

The local form and checkout-related routes are non-production review surfaces.
Use the live LawnPro site for real orders, live stock, and form delivery.
